## jumbo worker view

Display the current worker's identity and claim settings.

### Synopsis

```bash
> jumbo worker view
```

### Options

None.

### Behavior

Displays the current worker profile, including the worker identity and claim configuration. The worker identity is a unique identifier assigned to each LLM agent session, used for goal claim ownership and work tracking.

### Examples

```bash
> jumbo worker view
```
